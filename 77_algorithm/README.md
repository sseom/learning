
##알고리즘
- [헬로월드 : 알고리즘 문제](http://tryhelloworld.co.kr/challenges?language=javascript)
- 일정 : 한주에 1~2 문제
- 사용 언어 : javascript
- 문제 풀이 방식 :
    1. 손코딩 : 문제 해결의 방향 고민
    2. 코딩 
    3. 문제 해결후 다른 사람 코드 확인 
    4. 다른사람 코드 분석
    5. 정리해서 깃허브에 업로드
