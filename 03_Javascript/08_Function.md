#Function  함수

###함수를 만드는 자주 사용되는 방법 2가지
1. 함수를 정의하는 방법
    - `function fun(){ console.log('잼나'); }`
2. 함수 표현식을 변수에 참조하는 방법
    - `var funny = function(){ console.log('재밌네'); };`
    - 세미콜론 붙여야함.

###함수 호출
- 함수이름 호출(실행) `fun();`
    + undefind ==> return 값이 없어서 undefined 반환
- 함수를 참조!! `fun;` ==> 실행아님, 호출 아님
    + function
